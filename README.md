# Ai-Assisted-threat-Detection-Dashboard 
AI-Assisted Threat Detection Dashboard
Model Research
Objective

The objective of this model research is to identify the most suitable Artificial Intelligence (AI) and Machine Learning (ML) models for developing the AI-Assisted Threat Detection Dashboard. Various cybersecurity resources, research papers, technical blogs, and machine learning documentation were studied to understand which models are most effective for detecting cyber threats, classifying attacks, and predicting security risks.

The models were compared based on their learning type, cybersecurity use case, prediction capability, accuracy, and evaluation metrics such as Accuracy, Precision, Recall, F1-Score, and ROC-AUC.

Why These Models Were Chosen
Isolation Forest

Isolation Forest was selected because it effectively detects unusual network behavior and unknown cyber attacks without requiring labeled training data. It is widely used for anomaly detection in cybersecurity.

Random Forest

Random Forest was chosen because it provides high classification accuracy, reduces overfitting, and performs well on complex cybersecurity datasets.

XGBoost

XGBoost was selected because it offers excellent predictive performance, high accuracy, and efficient risk scoring, helping security teams prioritize critical threats.

Decision Tree

Decision Tree was included because it is simple, easy to interpret, fast to train, and useful for classifying different types of cyber attacks.

Conclusion :
Based on the research, Isolation Forest, Random Forest, XGBoost, and Decision Tree were identified as the most suitable machine learning models for the AI-Assisted Threat Detection Dashboard.

Isolation Forest detects anomalous network activities.
Random Forest accurately classifies cyber threats.
XGBoost predicts threat severity and prioritizes security incidents.
Decision Tree provides fast and interpretable attack classification.

These models improve cybersecurity monitoring, enhance threat detection accuracy, and support faster incident response, making them suitable for developing an AI-powered threat detection dashboard.
